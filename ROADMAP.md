# Wyrm Roadmap

## Vision
Become the standard memory layer for AI assistants, enabling persistent context across conversations and tools.

---

## Current: v3.0.0 (February 2026)

### Features
- ✅ MCP server with full tool support
- ✅ SQLite with WAL for performance
- ✅ Project/Session/Quest/Context storage
- ✅ HTTP API with authentication
- ✅ Resilience with auto-recovery
- ✅ Data lake for large datasets
- ✅ CLI with interactive shell
